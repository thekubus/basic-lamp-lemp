Welcome to the basic-lamp-lemp-cookbook wiki! This wiki is for the cookbook to cook basic Linux server, with either Apache or NGINX, MySQL, and PHP-FPM.

Recipe

  • httpd-apache. This recipe will install Apache 2 HTTP Server. Please use either Apache or NGINX, not both
  • httpd-nginx. This recipe will install NGINX HTTP Server. Please use either Apache or NGINX, not both (Future development)
  • db-mysql. This recipe will install MySQL server
  • mysql-createdb. This recipe will create new MySQL database (Future development)
  • php-fpm. This recipe will install PHP FPM
  • httpd-user. This recipe will create user for httpd (NGINX or Apache)
  • fpm-user. This recipe will create user for php-fpm

Attribute

  • default['basic-lamp-lemp']['httpd-user'] = The user to be used by httpd daemon
  • default['basic-lamp-lemp']['httpd-group'] = The group to be used by httpd daemon
  • default['basic-lamp-lemp']['httpd-instance'] = The name of Apache httpd instance to configure. Refer to chef httpd cookbook for explanation of instance
  • default['basic-lamp-lemp']['server_admin'] = The email address of Server Administrator, to be used in httpd 404 page
  • default['basic-lamp-lemp']['document_root'] = The root directory of httpd daemon
  • default['basic-lamp-lemp']['passwords']['secret_path'] = MySQL root password secret path. Not being used at the moment
  • default['basic-lamp-lemp']['php-fpm-user'] = The user to be used by php-fpm daemon
  • default['basic-lamp-lemp']['php-fpm-group'] = The group to be used by php-fpm daemon

Limitations

  • Only tested on Ubuntu platform. Might works on other Linux platform. Feel free to port, test, and let me know the result
