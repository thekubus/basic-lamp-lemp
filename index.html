This is the recipe to cook basic LAMP/LEMP Server. That means either Apache or NGINX,

Welcome to the basic-lamp-lemp-cookbook wiki! This wiki is for the cookbook to cook basic Linux server, with either Apache or NGINX, MySQL, and PHP-FPM.

Recipe

  • httpd-apache. This recipe will install Apache 2 HTTP Server. Please use either Apache or NGINX, not both
  • httpd-nginx. This recipe will install NGINX HTTP Server. Please use either Apache or NGINX, not both
  • httpd-configure. This recipe will configure new sites in httpd server
  • mysql-install. This recipe will install MySQL server
  • mysql-createdb. This recipe will create new MySQL database
  • php-fpm-install. This recipe will install PHP FPM
  • httpd-user. This recipe will create user for httpd (NGINX or Apache)

Attribute

  • default['basic-lamp-lemp']['httpd-user'] = The user to be used by httpd daemon
  • default['basic-lamp-lemp']['httpd-group'] = The group to be used by httpd daemon
  • default['basic-lamp-lemp']['document_root'] = The root directory of httpd daemon

Limitations

  • Only tested on Ubuntu platform. Might works on other Linux platform. Feel free to port, test, and let me know the result, or update this Wiki
